# MVP Scope (v1)

## In Scope
- Lead intake (webhook + manual entry)
- Lead inbox with statuses: new, contacted, booked, closed
- **Script customization from day one**
  - template editor for first response
  - ON/OFF toggle
  - variable placeholders (`{{name}}`, `{{business_name}}`, `{{booking_link}}`)
  - live preview before save
- Business settings: name, timezone, hours, booking link
- Daily digest summary (Discord/email)

## Out of Scope (v1)
- Multi-location + advanced team permissions
- Stripe billing
- Gmail parsing
- Multi-step AI conversation agent
- Advanced workflow builder

## Success Metrics
- Median first-response time < 5 minutes
- Positive pilot feedback on usability
- Measurable booked-call lift during pilot
