# Product Requirements (v1)

## Core Outcome
Increase booked calls from inbound leads by reducing response time.

## Primary User
Business owner with low technical skill.

## Must-Have Features
1. 5-step onboarding wizard
2. Lead inbox with statuses (new, contacted, booked, closed)
3. Template editor (first response + follow-ups)
4. Auto-reply ON/OFF toggle
5. Business settings (hours, timezone, service area)
6. Daily digest report

## Integrations (v1)
- Web form webhook
- Gmail forwarding/parser
- Calendly booking link
- Stripe subscription

## Success Metrics
- Median first-response time < 5 min
- Lead-to-booking conversion improvement
- Weekly active business accounts
